# 老师版九上预习包设计

## 背景

当前应用已经能展示五科章节、英语单词和本地进度，但内容形态仍偏原型：

- 语文课文目录只做了简略覆盖，每课默写词多为 4-5 个，不足以支撑九上预习。
- 数学、物理、化学仍有一部分题目像“学习方法提示”，不像江苏初三真实练习。
- 每课页面还有历史遗留的闭环、诊断、回忆提示等模块，学习价值不够集中。

本轮按用户选择的 B 方案执行：做成“老师版九上预习包”。第一轮目标不是做完整题库平台，而是让孩子打开任意一课，都能看到清楚的知识讲解和少量高质量练习。

## 目标

1. 每科按教材章节展示完整九上预习目录。
2. 每课形成稳定结构：知识点讲透 + 少量练习 + 答案解析 + 易错提醒。
3. 语文补齐课文级预习资料，并提供可展开/收起的“原文”区域；公版古诗文可显示全文，现代课文只显示已授权或用户提供的文本。
4. 理科题目改成真实题型：选择、填空、计算、实验探究、综合应用。
5. 英语保持简单：单词、重点短语、句型、知识点讲解、小自测。
6. 孩子端默认只看到已校对内容；待校对内容不进入主学习路径。

## 非目标

- 不做账号、班级、家长后台、支付和社区。
- 不抓取或批量复制未授权的现代课文全文；需要全文时，使用公版文本、用户提供文本或已授权来源。
- 不批量收录未确认授权的历年真题；可录入用户提供或明确允许转载的真题原题，其余用“真题考法参考”的原创变式。
- 不在第一轮覆盖九下或中考总复习。
- 不做实时 AI 家教对话。

## 教材与来源

默认地区仍按江苏常州：

- 语文：统编/人教版九年级上册。
- 数学：苏科版九年级上册。
- 英语：译林版九年级上册。
- 物理：苏科版九年级上册。
- 化学：沪教版九年级上册，优先按用户确认的沪教版和当前 2024 版目录结构校对。

目录和知识点允许人工后续修订。内容原则：

- 课文标题、作者、文体、学习要点可以展示。
- 课文原文放在默认收起的“原文”区：古诗文等公版文本可直接展示；现代课文默认只展示导读和关键学习点，除非文本来源已授权或由用户提供。
- 题库支持两类来源：`sourceType: "authorized-original"` 的授权真题原题，和 `sourceType: "original-variant"` 的原创变式。
- 对不确定版本的章节，用 `reviewStatus: "pending"` 或来源备注标出，不进入孩子默认路径。

## 页面结构

### 学科页

学科页只保留两层：

1. 顶部章节切换。
2. 当前章节的“课列表 + 当前课内容”。

不再默认展示学习闭环总览、入口诊断、路线进度、覆盖检查等管理型内容。管理型信息只保留在开发/家长模式或隐藏详情中。

### 每课页面

每课固定显示：

1. 预习目标：3-5 条，孩子知道今天学什么。
2. 知识点讲解：每个知识点含“是什么、怎么用、易错点”。
3. 本课必会：语文字词/文言点，数学公式，物理概念与公式，化学实验与符号。
4. 小练习：3-6 题，按题型分组。
5. 答案解析：默认折叠，点击展开。

每课默认不要出现重复提示卡，例如“先写依据再选择答案”“会了/还不会”这类通用模块。自评按钮只保留在真正需要记录掌握状态的地方。

所有讲解和提示必须像老师当面备课，而不是模板拼接。禁止整页反复出现“先确认条件”“写出依据”“真题考法参考”等空泛句式；每句话都要直接服务本课内容。

## 语文设计

语文不是只做 5 个词。每课至少包含：

- 课文标题、作者或出处、文体。
- 课前导读：100-180 字，说明读什么、抓什么。
- 必会词语：现代文 8-15 个；古诗文 10-20 个，含字音、字形、释义或语境。
- 阅读要点：3-5 个，如结构、人物、论证、意象、情感、语言。
- 阅读理解题：3 题，覆盖概括、赏析、主旨或论证思路。
- 写作迁移：1 题，把表达方法迁移到自己的材料。
- 原文区：默认收起；公版古诗文显示全文，现代课文显示“需授权文本”状态或用户导入后的全文。

古诗文额外包含：

- 文言实词。
- 重点虚词或句式。
- 翻译检查点。
- 背诵默写提示。

显示方式：

- 章节卡显示本单元所有课文标题和词语数量。
- 当前课显示完整预习包。
- 原文不抢占页面主内容，统一放在“展开原文”折叠区。
- 没有授权全文时，折叠区只显示“暂不展示全文，可使用课本阅读原文”，并继续展示导读、词语和题目。

## 英语设计

英语继续保持简单，不恢复复杂每日任务：

1. 单词：按课本 Wordlist 展示全单元词/短语，支持三种模式：
   - 英文显示，悬停/点击看中文。
   - 中文显示，悬停/点击看英文。
   - 英文和中文都显示。
2. 读音：每个词保留小喇叭按钮，使用浏览器语音。
3. 重点短语：展示英文 + 中文释义，短语可读音。
4. 句型：列出结构、例句、中文理解和仿写提示。
5. 知识点讲解：每单元 3-5 个语法/表达点。
6. 小自测：5 题以内，答案折叠，不做复杂记录。

英语页面不要显示每日单词预习路线、重复回忆提示或大块进度闭环。

## 数学设计

数学题要接近江苏初三练习风格，不再问“第一步应确认什么”。每章按知识点出题：

- 基础选择：概念辨析、公式条件。
- 填空：计算结果、参数范围、几何角度。
- 计算题：完整过程题。
- 应用题：一元二次方程、二次函数、概率统计等真实情境。
- 综合题：章节内 2 个以上知识点联动。

每题包含：

- 题干。
- 选项或作答提示。
- 答案。
- 关键步骤。
- 易错点。
- 难度：基础、巩固、提升。
- 来源类型：授权真题原题或原创变式。

第一轮重点升级数学第 1 章和第 5 章，因为这两章最能体现真实题型质量。

## 物理设计

物理按“概念 + 图示/实验 + 公式 + 题型”组织：

- 概念辨析题：如功、功率、机械效率、内能、电阻。
- 公式计算题：单位换算清楚。
- 实验探究题：器材、步骤、现象、结论、误差。
- 图像/电路/杠杆题：强调图示和条件。
- 综合应用题：生活情境转化为物理模型。

每课至少 4 题：

- 1 道概念选择。
- 1 道填空或判断。
- 1 道计算/作图。
- 1 道实验或综合题。

## 化学设计

化学按“宏观现象、微观解释、符号表达、实验规范”组织：

- 选择题：概念辨析、性质用途、实验安全。
- 填空题：化学符号、现象描述、仪器或操作。
- 实验题：制氧、空气成分、水净化、质量守恒等。
- 计算/推断题：相对分子质量、质量守恒、方程式基础计算。
- 综合题：现象 -> 依据 -> 结论/符号表达。

题目必须具体，例如：

- 可以问“红磷燃烧测空气中氧气含量，水面上升不足 1/5 的原因”。
- 不问“判断化学研究什么时最可靠依据是什么”这类空泛问题。

第一轮重点升级第 1-2 章，因为它们是孩子入门化学的核心。

## 数据结构调整

保留当前 `subject -> chapters -> lessons` 结构，但补充字段：

```js
{
  title,
  intro,
  mustKnow,
  explanations,
  practiceSets,
  reviewStatus,
  sourceNote
}
```

语文 `mustKnow` 示例：

```js
{
  dictationWords: [{ word, pinyin, meaning, note }],
  classicalChinese: [{ text, meaning, usage }],
  readingFocus: [],
  writingTransfer
}
```

理科 `practiceSets` 示例：

```js
{
  type: "choice" | "blank" | "calculation" | "experiment" | "comprehensive",
  difficulty: "基础" | "巩固" | "提升",
  sourceType: "authorized-original" | "original-variant",
  prompt,
  choices,
  answer,
  steps,
  trap,
  sourceNote: "原创变式 · 真题考法参考"
}
```

## 校对规则

新增测试和内容检查：

- 语文每课必须有标题、导读、至少 8 个默写词或文言点、阅读题和写作迁移。
- 语文现代课文不得内置未授权全文；公版古诗文和用户提供文本可在折叠区显示。
- 数学、物理、化学不得出现“第一步应先确认什么”“最可靠依据是什么”这类模板题。
- 内容不得大段重复“写依据、看条件、做迁移”之类通用提示；同一页面重复表达要改成具体知识点或删掉。
- 理科每课至少 3 种题型。
- 每题必须有答案、解析和易错点。
- 真题原题必须有 `sourceType: "authorized-original"` 和来源说明；否则不能标成真题原题。
- 孩子端不展示 `reviewStatus !== "approved"` 的内容。

## 第一轮实施范围

第一轮不追求一次填满所有真实题库，先做质量拐点：

1. 语文九上目录完整化，每课词语和导读补到可用水平。
2. 语文增加“展开原文”折叠区：先放公版古诗文全文，现代文保留授权占位。
3. 数学第 1 章、第 5 章题库升级为真实题型。
4. 物理第 11 章、第 14 章题库升级。
5. 化学第 1 章、第 2 章题库升级。
6. 清理页面里的通用模板模块，只保留核心学习内容。
7. 增加自动测试防止模板题和未授权全文回流。

## 验收标准

- 打开任意语文课，可以看到课文标题、导读、8 个以上词语/文言点、阅读题和作文迁移。
- 古诗文课可以展开查看公版原文；现代课文有折叠原文入口但不内置未授权全文。
- 打开数学/物理/化学重点章节，练习题像真实初三题，而不是学习方法问答。
- 页面中不再反复出现模板式提示，重复的“先写依据/先看条件”应被删掉或改成具体讲解。
- 手机端每课内容单列展示，不出现大面积空白或左右错位。
- 英语页面仍保持简单清晰。
- `node --test tests/*.test.mjs` 通过。
- 线上 GitHub Pages 能加载最新版本。
